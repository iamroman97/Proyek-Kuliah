# Ide Skripsi
Sistem Pendukung Keputusan untuk Menentukan Pemilihan Rumah Terbaik Menggunakan Metoda SAW

Metode SAW merupakan singkatan dari kata “Simple Additive Weight”. Metode SAW merupakan salah satu metode “Multiple Attribute Decision Making (MADM)”. Sistem Pendukung Keputusan metode Simple Additive Weighting (SAW) sering juga dikenal istilah metode penjumlahan terbobot. Konsep dasar metode SAW adalah mencari penjumlahan terbobot dari rating kinerja pada setiap alternatif pada semua atribut (Fishburn, 1967)(MacCrimmon, 1968).

Proses kerja sistem pendukung keputusan metode SAW :

- Menentukan Kriteria-kriteria untuk pemilihan beberapa alternati.
- Menentukan nilai bobot (W) untuk masing-masing kriteria (K).
- Menentukan Nilai Normalisasi Matriks ‘X’ (Rij).
- Menentukan Nilai Perferensi (Vi).
- Rumus normalisasi Matriks (Rij).
